# APTI Attendance - Google Form Status Checker

This Hugging Face Space monitors the status of a Google Form to check if it's accepting responses.

## Features

- Real-time form status checking
- Automated monitoring capability
- Simple API interface

## Configuration

The space checks the following form:
- **Form Title**: APTITUDE CLASS ATTENDANCE 2027 BATCH
- **Status**: Monitors if form is open or closed

## Usage

The space provides a Gradio interface to check the form status on-demand.
