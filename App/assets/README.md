# Default Alarm Sound

## IMPORTANT

A default `alarm.mp3` file is required in this folder for the app to work.

### Option 1: Add your own
- Place any MP3 alarm/ringtone file here
- Rename it to `alarm.mp3`

### Option 2: Download a free alarm sound
- https://freesound.org (search for "alarm")
- https://mixkit.co/free-sound-effects/alarm/
- https://www.zapsplat.com (free with account)

### Notes
- Users can choose their own alarm sound in the app
- This file is only used as a fallback default
- Keep the file size small (under 1MB recommended)
